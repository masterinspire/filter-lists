# Filter Lists

## About

Custom filter lists. Can be used in [uBlock Origin](https://github.com/gorhill/uBlock).

  * `filter-lists.txt`: main file.
  * `filter-lists-breaking-*.txt`: use at your own risks. may break some websites.
